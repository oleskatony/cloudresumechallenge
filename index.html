AWS Cloud Resume Challenge

This website was created as part of the Cloud Resume Challenge, an exciting initiative designed to highlight skills and explore various aspects of cloud computing. This project emphasizes the abilities required for a successful career in the cloud industry. For this endeavor, I focused on utilizing Amazon Web Services (AWS) as my preferred cloud provider. If you would like learn more about my attempt of the Cloud Resume Challenge, please check out my Github page that documents some of the components of this project.

This project has been an excellent learning experience that solidified my understanding of key cloud concepts. By applying Infrastructure as Code (IaC) principles with Terraform, implementing serverless solutions with DynamoDB and Lambda, and adopting CI/CD practices for website deployment, I have witnessed firsthand the efficiency and potential of AWS cloud. I highly encourage others to take on this challenge as it not only helps in developing cloud skills but also serves as a remarkable platform to showcase their abilities.
