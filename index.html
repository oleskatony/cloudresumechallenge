AWS Cloud Resume Challenge

As a demonstration of my AWS skills, I participated in the AWS Cloud Resume Challenge, which required me to create a resume website using a variety of AWS tools. The project involved setting up a custom domain name, implementing a visitor counter using Javascript, using Amazon DynamoDB as a database for the counter, creating an API using AWS API Gateway and Lambda to interact with the database, writing Python code for the Lambda function and setting up infrastructure as code using AWS Serverless Application Model (SAM) or Terraform, as well as implementing continuous integration and deployment using GitHub Actions.

The completed project showcases my ability to leverage multiple AWS services and technologies to build scalable and resilient web applications. If you would like to know more, please check out my Github where I go into detail about this challenge, as well as some other projects I have worked on.
